# Projet 1 Formation front-end
## Explication du TP
### Créer son planning, prendre en main sa formation

Ce premier projet à pour but de prendre en main la formation, afin de mettre en place un planning hebdomadaire, mais aussi un prévisionnel des dates de soutenances.
